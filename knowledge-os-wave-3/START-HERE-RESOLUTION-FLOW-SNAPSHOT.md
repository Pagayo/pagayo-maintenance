## Knowledge Resolution Flow

After boot, resolve task documentation through the topic registry — not loose Markdown paths.

1. Open `pagayo-vault/knowledge-os/03-registry/topic-registry.yaml`.
2. Find the matching topic for the task.
3. Resolve registry IDs via `pagayo-vault/knowledge-os/03-registry/canon-registry.yaml`.
4. Follow the resolution order defined in `pagayo-vault/knowledge-os/03-registry/RESOLUTION-RULES.md`.

Do not browse unregistered Markdown when a topic mapping exists.
